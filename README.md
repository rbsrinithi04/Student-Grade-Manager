# Student Grade Manager (Python)

A simple Python console application that allows users to add student details, record their marks, calculate averages, and automatically assign grades. This beginner-friendly project demonstrates the use of lists, loops, functions, and conditional logic in Python.

## Features
- Add student name and 3 subject marks  
- Automatically calculate average  
- Grade assignment based on average  
- View all student records  
- Simple text-based menu system  

## Grade Criteria
| Average Marks | Grade |
|---------------|--------|
| 90+           | A      |
| 75–89         | B      |
| 60–74         | C      |
| 40–59         | D      |
| Below 40      | F      |

1. Install Python 3.x  
2. Save the script as `student_grade_manager.py`
